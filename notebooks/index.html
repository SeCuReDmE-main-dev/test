    How To Use One Notebook¶

    Choose

    Select One Prompt

    Pick the notebook that matches the prompt you want to practice.

    Adapt

    Add Your Context

    Add topic, audience, and result wanted so the prompt becomes yours.

    Run

    Create A First Artifact

    Use the prepared prompt with your preferred assistant or classroom workflow.

    Reflect

    Keep The Useful Part

    Capture the result, what improved, and the next small step.

    Public Boundary¶

    These notebooks are learning worksheets. They do not need account keys, special setup, or private project details.
