# Plugin Registry

The controller registry records the surfaces it can route across.

Local plugin surfaces:

- `codex-memory-systeme`
- `securedme-cpanel-operator`
- `architecte-zero`
- `ffed-agent-memory-pack`
- `smarandache-book-forge`

Primary runtime surfaces:

- documents
- pdf
- spreadsheets
- presentations
- template creator

External connector surfaces:

- GitHub
- Gmail
- Google Calendar
- Google Drive

The controller does not copy connector secrets. It keeps connectors external and produces route plans.
