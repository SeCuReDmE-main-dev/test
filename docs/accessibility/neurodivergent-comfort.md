# Neurodivergent Comfort

SecuredMe Education treats accessibility as a public learning surface. The Access console supports comfort, attention, predictability, and user agency for neurodivergent readers without making medical claims.

The design follows public accessibility guidance: clear purpose, understandable structure, focus support, reduced memory burden, predictable navigation, and personalization. These controls support comfort; they do not replace accessibility review or user testing.

## Comfort Options

| Need | Helpful feature |
| --- | --- |
| Reduce distraction while keeping controls reachable | Focus Frame |
| Keep navigation predictable but less visually loud | Quiet Nav |
| Reduce sensory overload | Calm Layout |
| Reduce aggressive saturation | Stimulus Control |
| Help readers parse long pages | Content Chunking |
| Help users know where they are | Current Section |
| Support dyslexia and attention tracking | Reading Pacer |
| Support autism and anxiety through stable UI | Predictable Mode |
| Avoid metaphor-heavy labeling | Literal Labels |
| Help technical vocabulary discovery | Glossary Boost |
| Turn reading into visible progress | Checklist Mode |
| Mark progress in a familiar way | Done Marker |
| Reduce memory burden | Resume Point |
| Support ADHD activation | Sprint Timer |
| Support transition out of hyperfocus | Micro Breaks |
| Reduce pressure for anxious readers | Low Demand Mode |
| Support expert readers | High Agency Mode |
| Support motor comfort | Oversized Targets+ |

## Profiles

| Profile | Best first use |
| --- | --- |
| Neuro Focus | A reader gets distracted by navigation or decorative surfaces. |
| ADHD Sprint | A reader needs short activation loops, a timer, and a visible next step. |
| Autism Calm | A reader needs predictable labels, fewer stimuli, and low surprise. |
| Sensory Safe | A reader is overloaded by motion, saturation, or high-energy visuals. |
| Deep Work / Hyperfocus | A high-agency reader wants a quieter environment for long technical work. |

## Quick Check

- Open `Access` with the launcher and `Ctrl+U`.
- Confirm `Escape` closes the panel.
- Apply a profile and verify the page feels more comfortable.
- Start and stop the sprint timer with the button and `Ctrl+Alt+S`.
- Confirm dark/light mode remains separate from Access.
- Confirm mobile layout has no overlap and targets remain usable.
