## First 15 Minutes

| Minute | Action | Result |
| ---: | --- | --- |
| 0-2 | Choose one track above. | You know where to start. |
| 2-5 | Open the 40-prompt matrix and pick one prompt. | You have a clear request ready. |
| 5-10 | Replace the topic with your real learning goal. | The prompt belongs to your situation. |
| 10-14 | Read the answer and keep the useful part. | You have a first artifact. |
| 14-15 | Choose the next action. | Progress continues instead of stopping. |

## Notebook Path

Use the notebook version when you want the prompts in a worksheet format. The notebook works as a guided learning canvas:

1. open the notebook;
2. choose a prompt number;
3. add your topic, audience, and wanted result;
4. copy the prepared prompt;
5. paste the result back into your notes.

Open [Notebook / Colab](../notebooks/index.md) for the Jupyter file and the Colab link.

## How To Use A Prompt

Copy the prompt, then add three details:

- your topic
- your audience
- the result you want

Example:

```text
Create a 15 minute first activity using one SecuredMe Education tool.
Topic: visual algorithms
Audience: high school beginners
Result wanted: a small worksheet with one example and one reflection question
Use simple words, show the expected result, and finish with one next action.
```

## Good First Outputs

| Output | When To Use It |
| --- | --- |
| Checklist | When the learner needs visible progress. |
| Worksheet | When a teacher needs classroom material. |
| Comparison table | When the learner must choose between options. |
| Reflection note | When the learner needs to understand what changed. |
| Build steps | When the result should become a small project. |
