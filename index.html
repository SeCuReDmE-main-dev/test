    Tier 4: Education

    Learn safely. Build clearly.

    Apprendre, verifier, fabriquer. A public path for students, teachers, and builders using the SecuredMe Education suite.

    Learn / Apprendre Connect / Connecter Explore / Explorer Collaborate / Collaborer

    Start Here / Commencer Ici¶

    This public Education hub helps new users learn the SecuredMe plugin suite with visible progress, safe evidence, and reusable prompts.

    It is designed for two audiences at the same time:

    • students and young builders who need a concrete first week of progress;
    • professors, mentors, and operators who need readable evidence and clear boundaries.

    What You Can Do First¶

    Path Start here Result
    Learn / Apprendre Capability Matrix Understand the 40 plugin capability tests.
    Connect / Connecter Education Controller See how controller, cPanel, GitHub, docs, and sessions route together.
    Explore / Explorer Runbooks Follow the first-week onboarding path.
    Collaborate / Collaborer Prompting Guide Fork one prompt into your own workflow.

    Core Question¶

    What can the SecuredMe Education suite do safely, what evidence proves it, and how can a new user learn to build their own workflow?

    Documentation Areas¶

    • Capability Matrix: 40 public plugin capability tests and reusable prompts.
    • Wiki: navigation, documentation standards, and source map.
    • Tests: executed and planned test records.
    • Education Mesh: suite boundaries, mesh contracts, and readiness.
    • API Surface: cPanel, DNS, SSL, email, deployment, and suite mapping surfaces.
    • Runbooks: repeatable operating procedures.
    • Security: redaction and public boundary rules.
    • Status: current repository and plugin state.

    Operating Boundary¶

    This is public documentation. It must not contain credentials, raw .env material, cPanel session URLs, private keys, mailbox passwords, private cPanel exports, or deployment archives.

    Live cPanel mutations require explicit authorization and must be documented as redacted public evidence.
