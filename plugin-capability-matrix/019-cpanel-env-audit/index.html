    #19: cPanel env audit¶

    Purpose¶

    Learn how cPanel work is checked in live-read mode without exposing credentials.

    Capability¶

    Field Value
    Plugin owner securedme-cpanel-operator
    Target modele .env envelope
    Live mode live-read
    Expected command or tool /securedme-education-controller:test cpanel-env-audit
    Expected evidence env audit reports required keys without values

    Acceptance¶

    • Pass: The expected evidence is produced, redacted, and tied to the named target.
    • Fail: The tool or artifact runs but contradicts the expected evidence or leaks unsafe material.
    • Blocked: The target, credential, local service, or connector is unavailable and the exact blocker is recorded.

    Reuse¶
